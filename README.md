# OpenPlayMatchMaker-app
2v2 Pickleball Open Play Match Maker App

# ==========================================
# Created with assistance from Gemini AI
# Prompter: Zimzee19
# Description: 
Developed by prompting Gemini 2.5 Pro to generate the core functionality, followed by manual refinement and integration.

Using common open play match making rules:
1. Played in order of first come first serve basis
2. Winner stay and split
3. Two (2) maximum consecutive games regardless of win

Functionality:

Ability to add and remove players from the player list and automatically updating the queue.  
Ability to input match result and submit result.    
"Next In Line" queue list.  
Ability to see match history.  
Ability to see invdividual player's match W-L tally at session end.  
Ability to update players in match    
Ability to rearrange queue  
Ability to edit erroneous match result and re-pair accordingly    

  3 Modes available
